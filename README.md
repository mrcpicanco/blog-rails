# Praticando: Ruby, ruby on rails e postgresql.
## TECH
- Ruby - 3.1.2
- Rails - 6.1.6.1
- Postgresql - 14
- Boostrap - 5.1
- Node - 16.16.1
- Yarn - 1.22.19


## Iniciar a aplicação

```sh
git clone git@github.com:mrcpicanco/blog-rails.git
cd blog-rails
rails s
```

## Acessando o servidor

```sh
localhost:3000
```
